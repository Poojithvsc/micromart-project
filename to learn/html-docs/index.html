Welcome to MicroMart

A comprehensive e-commerce platform built with microservices architecture, demonstrating enterprise Java patterns with Spring Boot.

What is MicroMart?

MicroMart is a fully-functional online shopping platform similar to Amazon or Flipkart. It allows users to register, browse products, place orders, and track deliveries. What makes it special is how it's built - using microservices architecture, where the application is split into small, independent services.

Why Microservices?

Instead of one large application (monolith), MicroMart is composed of 6 independent services. This allows each service to be developed, deployed, and scaled independently. If one service has a bug, the others continue working!

Key Features

  • User Authentication - Secure JWT-based login with role-based access control
  • Product Catalog - Full CRUD with search, filtering, and categorization
  • Inventory Management - Real-time stock tracking with optimistic locking
  • Order Processing - Complete order lifecycle from creation to delivery
  • Event-Driven - Asynchronous communication via Kafka
  • Service Discovery - Dynamic service registration with Eureka
  • API Gateway - Centralized routing, authentication, and rate limiting
  • Image Storage - Product images stored in AWS S3
